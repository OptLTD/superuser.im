# Superuser 介绍

superuser是一个简单易用的业务平台，让您根据业务场景轻松构建出符合自己诉求的业务系统。

## 设计理念

1. 简单易用
    superuser 开箱即用，无需进行复杂的配置，即可满足大部分业务场景的需求。
2. 灵活可扩展
    superuser 设计了一个灵活的扩展框架，可以通过应用、插件化方式轻松扩展功能。
3. 高效智能

## 功能特性

- 直观的图形用户界面
- 支持多种数据库连接
- SQL 查询编辑器与自动补全
- 数据导入/导出功能
- 数据库结构可视化
- 用户权限管理

## 数据库支持

Superuser 支持以下主流数据库:

- MySQL
- PostgreSQL
- SQLite
- Microsoft SQL Server
- Oracle Database
